## Installation of Node.js and npm
- Visit https://nodejs.org

- Download LTS version

- Run .msi installer

- Follow default installation steps

- Open Command Prompt

- Verify installation:
```bash
node -v
npm -v
```
---
## Installation of Angular CLI
Open Command Prompt

Run command:
```bash
npm install -g @angular/cli
```
Verify Angular installation:
```bash
ng version
```
---
## Steps to Create Angular Application

Create a new folder

Open Command Prompt inside folder

Create Angular app:
```bash
ng new myApp
```
Go inside project folder:
```bash
cd myApp
```
Start Angular server:
```bash
ng serve
```
Open browser and type:
```bash
http://localhost:4200
```
